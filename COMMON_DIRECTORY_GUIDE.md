## How to Use

### Accessing the Directory

1. Login as **Resident**
2. Click **"Common Directory"** in sidebar
3. Browse or search for service providers

### Adding a Service Provider

#### Manual Addition

1. Click **"Add Service Provider"** button
2. Fill in the form:
   - **Name*** (Required)
   - **Category*** (Required) - Select from dropdown
   - **Mobile Number*** (Required)
   - **Email** (Optional)
   - **Address** (Optional)
   - **Services Offered** (Optional)
   - **Description** (Optional)
3. Click **"Add Provider"**
4. Provider appears in directory

#### Automatic Suggestion (Smart Feature)

When you approve a visitor with service-related purpose:

1. **Approve visitor** in Pending Approvals
2. **System detects** service-related purpose
3. **Prompt appears**: "Add to Common Directory?"
4. **Review pre-filled** information
5. **Add or Skip** as desired

- Plumber, Electrician, Carpenter, Painter

- Repair, AC, Appliance
- Internet, Cable, Doctor, Nurse
- Tutor, Teacher, Healthcare, Medical
- And more...

### Rating a Provider

1. Find the provider card
2. Click **"Rate"** button
3. Select star rating (1-5):
   - 1 star = Poor
   - 2 stars = Fair
   - 3 stars = Good
   - 4 stars = Very Good
   - 5 stars = Excellent
4. Optionally write a detailed review
5. Click **"Submit Rating"**
6. Your rating is added and average is updated

### Viewing Reviews

1. Click on review count (e.g., "(5 reviews)")
2. See overall rating summary with large star display
3. View all individual reviews showing:
   - Reviewer name and avatar
   - Star rating
   - Review text
   - Date posted
4. Reviews sorted by most recent first

### Sharing a Provider

1. Find the provider card
2. Click **"Share"** button
3. If device supports native sharing:
   - Choose app (WhatsApp, SMS, Email, etc.)
   - Share provider details
4. If not supported:
   - Details copied to clipboard
   - Paste anywhere to share

### Verifying a Provider (Admin Only)

1. Login as Administrator or Superadmin
2. Find the provider card
3. Click **"Verify"** button (only visible to admins)
4. Provider gets green "Verified" badge with award icon
5. Verification info stored (who verified and when)

### Editing a Provider

1. Find the provider card
2. Click **"Edit"** button
3. Update information
4. Click **"Update Provider"**

### Deleting a Provider

1. Find the provider card
2. Click **"Delete"** button
3. Confirm deletion
4. Provider removed from directory

## FAQ

**Q: Who can add to the directory?**  
A: All residents can add service providers.

**Q: Can I add providers from outside the society?**  
A: Yes, as long as they serve the community.

**Q: What if I add wrong information?**  
A: You can edit or delete your entries anytime.

**Q: Can other residents edit my entries?**  
A: Currently yes, as it's community-shared. Future: only creator can edit.

**Q: How do I know if a provider is good?**  
A: Check their average rating, read reviews from other residents, and look for the verified badge.

**Q: Can I rate a provider multiple times?**  
A: Currently yes, but best practice is to rate once after using their service.

**Q: What if I disagree with a review?**  
A: Add your own review with your experience. Multiple perspectives help everyone.

**Q: Can I add the same provider twice?**  
A: Avoid duplicates. Search first before adding.

**Q: What if provider's number changes?**  
A: Edit the entry with new number.

**Q: Can I add multiple numbers for one provider?**  
A: Add primary number. Use description for additional contacts.

**Q: Is this visible to other societies?**  
A: No, each society has its own directory.

**Q: Can I export the directory?**  
A: Not currently. Future feature planned.
