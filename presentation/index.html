  • Resident selection for visit destination.
  • Real-time sync to resident's device.

Operation Flow: Summary

Arrival: Visitor arrives; Security captures photo & details.

Technical Foundation

  • Modern UI: Built with React + Vite for lightning-fast performance.
  • Real-time Sync: Cloud database (Supabase/Firebase) ensures gate & resident sync.
  • Offline Resilience: LocalStorage fallback keeps gates operational.
  • Secure: Multi-tier authentication and role-based access.
